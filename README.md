# 基于图像复原与生成式对抗网络的模糊人脸识别研究与仿真

## 环境配置

创建新的 python 环境并激活:
``` bash
conda create -n bfs python=3.11
conda activate bfs
```

下载依赖:
``` bash
pip install requirements.txt
```

## 项目结构

1. 'README.md': 本项目说明文件

2. 'requirements.txt': 当前 pip 列表快照

3. 'configs/': 项目中所有 yaml / json 超参

4. 'data/': 原始数据 & 流水线中间产物

5. 'models/': 模型定义（不含训练脚本）

6. 'utils/': 工具脚本

7. 'scripts/': 主要代码

8. 'logs/': 日志输出目录

## 模型和方法

### 1. 基于小波变换和卷积神经网络的模糊人脸识别方法

先用小波分解把模糊图的高频噪声拿掉并保留边缘，再把去噪后的图送进轻量级 CNN 提取特征、做分类；整个流程无需高分辨率复原，只利用小波的频域去噪能力和 CNN 的判别能力，即可在模糊图像上快速完成身份识别。

### 2. 基于最小二乘方滤波的 SRGAN 方法

最小二乘方滤波在频域做反卷积，快速削掉整幅图的运动模糊和噪声； 把初复原图像送入 SRGAN 的生成器，用感知＋对抗损失再超分并补回高频人脸细节。传统“稳”与 GAN“细”串行，识别阶段无需改网络即可显著提升模糊人脸 ID 准确率。用预训练 FaceNet 识别人脸
